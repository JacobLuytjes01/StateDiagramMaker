# StateDiagramMaker
A program to make simple state diagrams.

# Three Examples of Output:

```
                 ┌-----------┐                                             
┌-----------┐    |           |                                             
|           |--->|           |    ┌------┐    ┌----------┐    ┌-------┐    
|   Start   |    | Long Word |--->| TEST |    |   TEST   |<---| Final |    
|           |<---|           |    └------┘    └----------┘    └-------┘    
└-----------┘    |           |        |             |                      
       ^         └-----------┘        |             |                      
       |                |             |             |                      
       |                └---------------------------┘                      
       |                              |                                    
       └------------------------------┘                                    
                                                                           
┌-------┐    ┌---------┐    ┌------┐    ┌-----┐
| Start |--->| Playing |--->| Lose |    | Win |
└-------┘    └---------┘    └------┘    └-----┘
                   |                        ^
                   └------------------------┘

┌-----┐    ┌---┐    ┌----------------------┐    ┌------┐
| One |    | 2 |    | Box with Three in it |    | four |
└-----┘    └---┘    └----------------------┘    └------┘
    ^                                               ^
    └-----------------------------------------------┘
```
